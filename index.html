ATS Resume Filter System

This project analyzes resumes using Natural Language Processing (NLP) to detect key patterns and predict selection outcomes based on keyword relevance. Built as a collaborative effort, the system leverages Python for NLP and Tableau for visualization. It evaluates how certain keywords affect a candidate’s chances of getting selected, helping recruiters filter resumes more efficiently and fairly.
